# Loyaltymaster Design System Archive

This folder contains historical project material that has been moved out of the
canonical workspace root during Phase 1 cleanup.

## Why This Exists

The Loyaltymaster design system is being consolidated into one human-readable
and AI-readable source of truth at the workspace root. Older attempts and handoff
bundles are preserved here so their useful context is not lost, but they are no
longer authoritative.

## Archived On

2026-05-18

## Archive Rules

- Do not import code, assets, or documentation from `_archive/` in new work.
- Do not treat archived folders as canonical design-system sources.
- Use the root-level files and folders instead: `READ_FIRST_AI.md`,
  `BRAND.md`, `DESIGN_SYSTEM.md`, `PREFLIGHT.md`, `assets/`, `registry/`,
  `tokens/`, `specs/`, `library/`, and `showcase/`.
- If an archived file contains a useful historical pattern, copy the relevant
  idea into the canonical system first and cite the source in the destination
  document.
