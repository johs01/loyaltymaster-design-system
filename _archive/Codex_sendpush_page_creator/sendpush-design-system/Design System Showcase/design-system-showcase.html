System map

The page is split the same way the design system is split.

Foundations document the token logic. Components capture the reusable interface pieces. Marketing sections show how those pieces compose into the live sendPUSH flow.

Brand rules

sendPUSH stays product-first.

Product name is always sendPUSH. Tone is direct, practical, and reassuring. Primary CTA remains the yellow pill with the orange hard edge.

Typography

Rodger leads, Onest explains.

Rodger handles H1, H2, H3, and stat values. Onest drives navigation, body copy, labels, eyebrows, form fields, and utility UI.

Color logic

Warm surfaces, sharp accents.

Yellow, orange, blue, and ink sit on peach, cyan, salmon, yellow, and light panel backgrounds. The result is warm, commercial, and unmistakably branded.

Interaction

Soft lift, crisp reveal.

Hover states are restrained. Scroll reveals use opacity plus a 24px rise. Focus rings stay visible, mobile targets stay generous, and motion respects user settings.
