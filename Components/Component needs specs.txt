# Primary Button

## Purpose

Use this for the main action in a section.

## Visual rules

- Use the primary Loyaltymaster brand color from `design.md`.
- Use the approved button radius from `design.md`.
- Use approved spacing from `design.md`.
- Text must be short and action-based.
- Do not create a custom button style.

## Allowed use

- Main hero action
- CTA section action
- Pricing card action
- Lead form submit action

## Not allowed

- Do not place next to another Primary Button.
- Do not use inside blog body text.
- Do not use in footer link groups.
- Do not use for low-priority actions.

## Content rules

Good labels:

- Start free
- Book a demo
- Create a pass
- Try SendPush

Bad labels:

- Click here
- Submit
- Read more
- More information

## Mobile rules

- Full width inside forms.
- Normal width inside hero sections unless the layout stacks.

## AI instruction

When the user asks for a main CTA, use `button.primary`.  
Do not invent a new button.  
Do not change the color, radius, padding, or hover style.
