# Runbook POC Fixtures

These fixtures are proof-of-concept scaffolding for future Runbook A and Runbook B authoring. They are not final runbooks.

- Runbook A fixture proves a page brief can become a strict markdown outline.
- Runbook B fixture proves a markdown outline can become a route-agnostic production page patch plan.
- End-to-end fixture proves failures can be diagnosed without hidden human interpretation.
