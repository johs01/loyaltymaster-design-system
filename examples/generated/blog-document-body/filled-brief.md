# Filled Brief: Blog/Document Body

Source template: `templates/blog-document.md`

## Artifact Type

Educational blog/document body for the existing Loyaltymaster production
website.

## Shell Scope

Body-only. The existing production website supplies the global chrome.

## Goal

Create a visual educational document body that explains how direct loyalty
messages help returning-customer businesses create measurable repeat visits.

## Audience

Business owners, marketing operators, and internal Loyaltymaster reviewers who
need a system-aligned educational page.

## Primary Takeaway

Direct loyalty messages work best when the visitor can understand the customer
moment, the approved process, the comparison, the proof, and one final action.
