# Loyaltymaster AI Agent Rules

Before creating or editing any UI:

1. Read DESIGN.md.
2. Read PRODUCT.md.
3. Read design-system/component-registry.json.
4. For every component used, read its rule file from design-system/component-rules/.
5. Do not create new visual styles unless asked.
6. Do not create new components if an approved component already exists.
7. Use approved React/Tailwind components.
8. Validate component pairings before writing code.
9. Use CSS variables and design tokens only.
10. If a rule conflicts, component-specific rules beat general rules.
